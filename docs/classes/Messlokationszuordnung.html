Class Messlokationszuordnung

Mit dieser Komponente werden Messlokationen zu Marktlokationen zugeordnet.

Dabei kann eine arithmetische Operation (Addition, Subtraktion) angegeben werden, mit der die Messlokation zum Verbrauch der Marklokation beiträgt.
